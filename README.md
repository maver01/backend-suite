# backend-suite

This repository contains a series of modules that can be used to build a backend.

Folder structure:

```
├── platform                        (e.g.: platform-web)
│   └── framework                   (e.g.: framework-spring-boot)
│   │   │   └──  project            (e.g.: project-web-receive-files-v1)
│   │   │   │   └──  source code    (e.g.: src)
```
